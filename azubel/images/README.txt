Place hero and gallery images in this folder.

Suggested filenames:
- hero-family.jpg (homepage hero)
- clinic-1.jpg
- team-1.jpg

Recommended sizes:
- Hero: 1600x900 (or similar 16:9), optimized ~150–250 KB.
- Other images: 1200px wide max, ~100–200 KB.

Tips:
- Use JPEG for photos, SVG/PNG for logos.
- Compress via https://squoosh.app/ or similar.
